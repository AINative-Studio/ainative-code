# API Reference

This directory contains API reference documentation for AINative Code.

## Provider Interfaces

Documentation for LLM provider implementations:

- **Anthropic Claude** - Claude 3.x models API
- **OpenAI** - GPT-4 and GPT-3.5 API
- **Google Gemini** - Gemini Pro and Ultra API
- **AWS Bedrock** - Claude on Bedrock API
- **Azure OpenAI** - Azure-hosted GPT models API
- **Ollama** - Local model inference API
