## Key Features

### Chat Mode
Interactive AI-powered coding assistance with streaming responses.

### Session Management
Save and resume your coding sessions.

### Multi-Provider Support
Use Anthropic Claude, OpenAI GPT, Google Gemini, and more.

### Platform Integrations
- **ZeroDB** - Vector and NoSQL database operations
- **Design Tokens** - Extract and sync design systems
- **Strapi CMS** - Content management integration
- **RLHF** - Submit feedback to improve AI responses
