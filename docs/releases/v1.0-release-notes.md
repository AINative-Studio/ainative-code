# AINative Code v1.0 Release Notes

**Release Date**: January 4, 2026

**AI-Native Development, Natively**

---

## Executive Summary

We are thrilled to announce the official **v1.0 release** of AINative Code, a next-generation terminal-based AI coding assistant that brings together the best features of modern AI development tools with native integration to the AINative platform ecosystem.

This milestone release represents months of development work, featuring comprehensive LLM provider support, a beautiful terminal UI, powerful platform integrations, and production-ready infrastructure. With over 90 completed tasks and thousands of lines of battle-tested code, AINative Code v1.0 is ready to transform your development workflow.

---

## What's New in v1.0

### 1. Multi-Provider AI Support

AINative Code supports **6 major LLM providers** out of the box:

- **Anthropic Claude** - Including Claude 3.5 Sonnet with streaming and extended thinking
- **OpenAI GPT** - GPT-4 and GPT-3.5 with full API support
- **Google Gemini** - Gemini Pro with Vertex AI integration
- **AWS Bedrock** - Claude and other models on AWS infrastructure
- **Azure OpenAI** - Azure-hosted OpenAI models
- **Ollama** - Local open-source model execution

**Key Features**:
- Automatic provider fallback for high availability
- Streaming responses with real-time display
- Prompt caching for 90% cost reduction
- Extended thinking visualization
- Per-provider configuration and rate limiting

### 2. Hybrid Authentication System

Enterprise-grade authentication with multiple auth flows:

- **JWT Authentication** - Secure token-based auth with automatic refresh
- **OAuth 2.0 PKCE Flow** - Secure authorization code flow
- **API Key Management** - Separate keys for each LLM provider
- **OS Keychain Integration** - Secure credential storage using macOS Keychain, Windows Credential Manager, Linux Secret Service
- **Token Auto-Refresh** - Automatic token refresh before expiration
- **Local Fallback** - Continue working when network is unavailable

**CLI Commands**:
```bash
ainative-code auth login          # Interactive login
ainative-code auth logout         # Logout and clear credentials
ainative-code auth refresh        # Manually refresh tokens
ainative-code auth status         # Check authentication status
```

### 3. ZeroDB Platform Integration

Complete integration with AINative's ZeroDB platform:

#### NoSQL Operations
```bash
ainative-code zerodb table create    # Create NoSQL tables
ainative-code zerodb table query     # Query with filters and pagination
ainative-code zerodb table insert    # Insert rows
ainative-code zerodb table update    # Update rows
```

#### Vector Operations
```bash

ainative-code zerodb vector search   # Semantic similarity search
ainative-code zerodb vector list     # List all vectors
ainative-code zerodb vector stats    # Get statistics
```

#### Agent Memory
```bash
ainative-code zerodb memory store    # Store conversation context
ainative-code zerodb memory retrieve # Semantic memory search
ainative-code zerodb memory list     # List memory history
ainative-code zerodb memory clear    # Clear old memories
```

## Upgrade Instructions

### Fresh Installation

#### macOS (Homebrew)
```bash
brew install ainative-studio/tap/ainative-code
```

#### macOS/Linux (Direct Download)
```bash
# macOS Apple Silicon
curl -LO https://github.com/AINative-studio/ainative-code/releases/latest/download/ainative-code-darwin-arm64
chmod +x ainative-code-darwin-arm64
sudo mv ainative-code-darwin-arm64 /usr/local/bin/ainative-code

# macOS Intel
curl -LO https://github.com/AINative-studio/ainative-code/releases/latest/download/ainative-code-darwin-amd64
chmod +x ainative-code-darwin-amd64
sudo mv ainative-code-darwin-amd64 /usr/local/bin/ainative-code

# Linux AMD64
curl -LO https://github.com/AINative-studio/ainative-code/releases/latest/download/ainative-code-linux-amd64
chmod +x ainative-code-linux-amd64
sudo mv ainative-code-linux-amd64 /usr/local/bin/ainative-code
```

#### Windows
```powershell
Invoke-WebRequest -Uri "https://github.com/AINative-studio/ainative-code/releases/latest/download/ainative-code-windows-amd64.exe" -OutFile "ainative-code.exe"
Move-Item ainative-code.exe C:\Windows\System32\
```

#### Docker
```bash
docker pull ghcr.io/ainative-studio/ainative-code:latest
docker run -it --rm ghcr.io/ainative-studio/ainative-code:latest
```

### Initial Setup

1. **Initialize configuration**:
   ```bash
   ainative-code init
   ```

2. **Authenticate with AINative** (optional):
   ```bash
   ainative-code auth login
   ```

3. **Configure LLM provider**:
   ```bash
   ainative-code config set provider anthropic
   ainative-code config set anthropic.api_key "your-api-key"
   ```

4. **Start using**:
   ```bash
   ainative-code chat
   ```
